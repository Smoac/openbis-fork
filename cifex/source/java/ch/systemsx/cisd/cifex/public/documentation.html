I forgot my password, how can I receive a new one?

If you are a temporary user, you should contact the person who created the account for you, to set a new password.

If you are a permanent user or don't know who created the account for you, you can contact the administrator of the system, which is listed on the footer of the website.

I have not received the mail with my login and password, were can I get it from?

First have a look in your spam filter, perhaps the mail is there.

If you can't access these mails or the mail is not there, you should contact the person, who wants to share the files with you or the on who created your temporary account. This person can reset your password. When you next time login to Cifex, please change this password!

If you are a permanent user or don't know who created the account for you, you should contact the administrator of the system, which is listed on the footer of the website.

I don't have the link to the file to download anymore. Where can I get it from?

You don't need the link to the file, you simply can login to Cifex and there you have a complete list with all files which are available for you.

If you don't have a password to access the system, please have a look on the FAQ entry "I have not received the mail with my login and password. Were can I get it from?"

How can I set the password of temporary users I created?

In the main view you have a list of users you have created. In the last column is a link with the value edit, which opens a new Window with a form to edit the user. If you leave the password field empty, the password is not changed and the last password will be kept. So be aware that you have to fill out the password field, to set a new password and don't forget to notify the user about the changes (Checkbox in the form).

How can I change a list of users with whom my file has been shared?

Use 'Edit sharing' link in the 'Files I have uploaded' section.

I want to share confidential files, what do I need to consider?

Email addresses are not unique in CIFEX, which mens that several users with the same email address may exist. In this case all of them would get the file you upload to that email address.

If you want to share your file only with some of the users assigned to given email address, you can specify the user name in the file upload form with the prefix 'id:'. So if you want to upload a file for user 'x' with the mail a@abc.com, you can specify 'id:x' in the Recipients field and only user 'x' will receive the file.

If you know just the email address but not the user name, you can add the email address to the Recipients field and click on 'Validate users' button. A window will appear and all the information about the users connected to the email addresses you entered will be displayed. You can add more users or select only those you want to share the file with.

Note that the person who registered an account is able to change it and this way may get access to the files shared with it. Check if the registrant is someone you trust. Should this not be the case - you'd better create manually a new temporary CIFEX account for your friend and use that new account's code to share your files.

A user with this email address already exists, but it is not the one I want to share the file with!

If you want to upload a file for an email address which already exists in Cifex but does not belong to the user you want to share the file with, you should create a new temporary user account with this email address. Thanks to the fact, that the mail addresses are not unique, this is possible. Now you can specify the user name in the upload field with the prefix 'id:' or just enter the email address and do 'Validate Users' and only select the temporary user you created.

I'm trying to log in using my LDAP/University/other external account, but the system does not let me in.

In certain configurations CIFEX allows logging in through external systems (like LDAP) as well as with internal authentication system (mostly used for temporary accounts). Not being able to log in using external username and password is a sign, that account with your username has already been taken by some temporary user. In such cases, please contact the administrator of your CIFEX installation.

I already have a CIFEX account, but would like to use the same password that I use for LDAP/University/other external account.

Just log in with your temporary password, click External authentication option and type your password.
