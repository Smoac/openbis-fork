This browser displays data registered in the BaSynthec project. Initially, data sets are displayed grouped by the technology used to produce them. An alternate display mode shows all strains for which data has been collected. In either mode, clicking on an entry opens a small window that allows downloading of the data in several different formats.

A third display mode, OD600, is for browsing of OD600 data.
