The High Content Screening Cell-Level Data API (HCSCLD) provides an efficient way to store and query cell-level result data from HCS measurements. It supports Image Segmentations, Object Tracking, Feature Values and Classification Results.

Classes to get started

In order to get started with the API, see the CellLevelDataFactory class.
